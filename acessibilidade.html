ACESSIBILIDADE

Medidas tomadas no código para garantir acessibilidade:

-Contraste de cores:

As cores do texto e do fundo foram escolhidas de forma a garantir um bom contraste, facilitando a leitura para usuários com deficiências visuais.

-Tamanho da fonte:

O tamanho da fonte foi definido de forma a garantir uma leitura confortável para a maioria dos usuários, facilitando a leitura para pessoas com dificuldades visuais.

-Botões e links acessíveis:

Os botões e links foram estilizados de modo a terem uma área de clique adequada, facilitando a seleção por usuários com dificuldades motoras ou que usam dispositivos de entrada alternativos.

-Foco visual:

Quando um botão ou link é selecionado, há um efeito visual que destaca o elemento, facilitando a identificação do foco de interação para usuários que navegam pelo teclado.

-Alternativas textuais para imagens:

Todas as imagens têm atributos alt definidos, fornecendo uma descrição textual do conteúdo das imagens. Isso é essencial para usuários que utilizam leitores de tela e para quem as imagens não são exibidas corretamente.

-Responsividade:

O layout foi projetado para ser responsivo, adaptando-se de forma adequada a diferentes tamanhos de tela e dispositivos, garantindo uma experiência consistente para usuários em diversas plataformas.
